# MangaReader

MangaReader is a modern, open-source manga reading platform designed for a seamless and high-quality reading experience.

## Features

- **Open Source**: Built with transparency and community contribution in mind.
- **Modern Tech Stack**: Developed using **React**, **Vite**, and **Tailwind CSS**.
- **Fast and Responsive**: Optimized for speed and works beautifully on any device.
- **Cloud Native**: Professionally hosted on **Cloudflare Pages** for high availability and performance.

## Hosting

This project is hosted exclusively on [Cloudflare Pages](https://pages.dev/).

- **Live URL**: [https://manga-readers.pages.dev](https://manga-readers.pages.dev)

## Tech Stack

- **Frontend**: React (with TypeScript)
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Deployment**: Cloudflare Pages

## Getting Started

To run the project locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Nepcoder0981/Manga-readers.git
    cd Manga-readers
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Start the development server**:
    ```bash
    npm run dev
    ```
